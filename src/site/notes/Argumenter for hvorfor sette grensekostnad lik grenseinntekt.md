# Argumenter for hvorfor sette grensekostnad lik grenseinntekt

Forklar hvorfor vi kan sette grensekostnad lik grenseinntekt når vi skal finne det største overskuddet.

## Fasit

$O'(x)=I'(x)-K'(x)$ 
$0=I'(x)-K'(x)\iff I'(x)=K'(x)$
