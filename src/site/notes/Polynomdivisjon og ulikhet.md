# Polynomdivisjon og ulikhet

Polynomet $P$ er gitt ved

$$P(x) = x^3 - 19x + 30$$

>[!oppgave]
>a) Utfør polynomdivisjonen $P(x) : (x - 2)$.
>b) Løs ulikheten $P(x) \geq 0$.
>c) Forkort brøken
>
>$$\frac{x^3 - 19x + 30}{x^3 - 2x^2 - 9x + 18}$$
