# Gjennomsnittlig vekstfart med program

Ada har laget programmet nedenfor.

```python ln
def f(x):
    return x ** 2 - 3 * x + 7

a = 0
b = 5

v = (f(b) - f(a)) / (b - a)

print(v)
```
