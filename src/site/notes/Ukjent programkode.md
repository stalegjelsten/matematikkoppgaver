# Ukjent programkode
En elev har skrevet følgende kode
```python
from math import sqrt            # importerer kvadratrotfunksjon
a = 0
b = 2
n = 10000

def f(x):
	return x**2 + 2

I = 0
h = (b - a)/n

for i in range(n):
	I = I + f(a + i*h)*h

print(round(I,3))
```

a) Forklar hva eleven ønsker å regne ut.

b) Hva blir det eksakte svaret på oppgaven eleven ønsker å løse?
