# Trekant med arealsetning og cosinussetning

Du får vite følgende om en trekant $ABC$

- $AB$ er 8
- $\angle A = 120\degree$
- Arealet av trekanten er $4\sqrt{3}$

>[!oppgave]
>Bestem lengdene av sidene $AC$ og $BC$ eksakt.
