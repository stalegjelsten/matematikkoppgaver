# Sum av aritmetisk rekke med kode

En elev har skrevet følgende kode:

```python ln
a = 3
d = 4

N = 10
S = 0

for i in range(N):
    S = S + a
    a = a + d

print(S)
```

>[!oppgave]
>a) Forklar hva eleven ønsker å regne ut.
>b) Hva blir resultatet når programmet kjøres, dersom N settes til 100 i linje 4?
